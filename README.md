# Scp-Web

## 本地环境

### `安装项目依赖`

`yarn install`

**_如果没有 yarn，Mac 可以使用 `brew install yarn`安装_**

### `启动项目`

`yarn start`

### `更新依赖`

`yarn upgrade-interactive --latest `

### 开发 IDE

- WebStorm (推荐)
- VS Code
